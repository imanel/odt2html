# ODT2HTML

## Usage

    $ odt2html [OPTIONS]

## Available options

Option          | Description
--------------- | -------------
--in [PATH]     | Path to input file(required)
--out [PATH]    | Path to output file(output to STDOUT if not provided)
--css [PATH]    | Path to css file(css in header if not provided)
--images [PATH] | Path to images directory
